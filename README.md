## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/username/react-tool.git
   cd react-tool
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```
